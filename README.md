## Dev setup

* clone repo
* run tox - in a virtualenv or with tox installed globally
  - tox -p auto          
  - tox -e lint
  - tox -e typecheck
  - tox -e py311
* add pre-commit
  - pip install pre-commit
  - pre-commit install
  - pre-commit run --all-files
